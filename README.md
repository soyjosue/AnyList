### Desarrollo

**Instrucciones de despliegue:**

1. Clonar el proyecto.
2. Copiar el archivo `.env.template` y renombrarlo a `.env`.
3. Ejecutar `yarn install`.
4. Levantar la base de datos con Docker: `docker-compose up -d`.
5. Levantar el backend de NestJS: `yarn start:dev`.
6. Visitar el sitio: [http://localhost:3000](http://localhost:3000).
